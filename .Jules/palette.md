## 2024-05-24 - Accessibility for Icon-only Buttons
**Learning:** The application uses Google Material Symbols (<span className="material-symbols-outlined">...</span>) for icons. When used inside icon-only buttons without text content, screen readers will incorrectly read the raw ligature text (e.g., "close", "chat", "send") unless `aria-hidden="true"` is applied to the span. Furthermore, these buttons frequently lack descriptive `aria-label`s and visible focus states for keyboard users.
**Action:** When working with icon-only buttons containing Material Symbols ligatures in this codebase, always manually add `aria-hidden="true"` to the inner span, an `aria-label` to the parent button, and visible focus outline styles (e.g., `focus-visible:ring-2`) to ensure full keyboard and screen reader accessibility.
